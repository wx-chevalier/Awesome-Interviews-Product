# 创想三维软件与固件核心被吐槽点汇总

## 一、软件类（以 Creality Print 切片软件为主）

### 1. 功能局限与易用性差

- **模型编辑能力弱**：打孔仅支持沿切面打，曲面竖直孔难以实现；克隆功能存在 Bug，操作体验卡顿（摘要 2）；模型检测修复流于形式，无法识别多数潜在问题（如面缺失、非流形结构），新手易踩坑（摘要 2）。

- **功能缺失与不合理**：不支持 Fusion360 打印，兼容性受限（摘要 2）；web 端上传模型时，即使选中“知识共享权”仍无法发布，功能逻辑矛盾（摘要 2）；无“关闭底面支撑（筏）”的直观设置，需手动调整“平台附着”参数，新手难理解（摘要 2）。

### 2. 稳定性与兼容性问题

- **上传与打印异常**：网络不佳时，模型上传进度会显示“100%以上”，且上传后打印易出现“自动打第二遍 → 归零撞模型死机”，同文件用其他切片软件无此问题（摘要 2）；创想 Print 保存的 3MF 文件，用 Snapmaker U1 切片软件打开后无法正常打印（打印头不归位、温度参数错误），仅重新导入 STL 可解决，格式兼容性差（摘要 6）。
- **文件管理混乱**：“打印列表”文件无法直接清空，需插入 TF 卡后通过 App 删除，操作繁琐（摘要 2）。

### 3. 生态与适配问题

- **跨设备体验差**：Mac 端切片软件频繁闪退，打印中无法排除模型；与打印机连接后易自动断开，需反复重连（摘要 6）；手机端 App 仅能查看“卡成 PPT”的摄像头画面，无实用控制功能（如暂停微调、参数修改）（摘要 6）。

## 二、固件类（覆盖 Ender3 S1 Pro、K2 Plus、CR-10 Max 等机型）

### 1. 功能失效与参数异常

- **新功能适配不全**：Ender3 S1 Pro 新固件“预览功能”部分用户无法使用，仅显示默认“小人图”；打印速度与切片设置脱节，起始层直接以全速打印，导致首层附着力差（摘要 1）。
- **核心参数不一致**：K2 Plus 打印同一份 G-code 时，从切片软件直连打印与从打印机内存打印的 Z-offset 完全不同（均设为 0 仍有偏差），导致首层高度异常（过厚/刮床）（摘要 3）；AI 打印检测功能频繁失效，无法识别打印失败（如堵头、翘边）（摘要 3）。

### 2. 稳定性与硬件协同问题

- **响应延迟与操作逻辑缺陷**：K2 Plus 固件存在“意外响应延迟”，紧急停止时无细分选项（仅单一“停止”，缺“立即停/中止任务/暂停”），用户无法根据场景快速处理（如断料时需暂停而非中止）（摘要 3）；闲置时挤出机电机不关闭，持续发热导致耗材软化堵头，需手动断电规避（摘要 3）。
- **硬件联动故障**：K2 Plus 的 CFS（送料系统）固件适配差，filament encoder 易故障，频繁触发 FB2846 报错、PTFE 管断开，临时调整仅能缓解，需更换编码器套件（摘要 3）；CR-10 Max 固件导致 Z 轴探头失效，无法归位打印，且自带 SD 卡文件半数不可用（文件名含特殊字符/超 16 字符不识别）（摘要 5）。

### 3. 升级与兼容性风险

- **固件更新困难**：CR-10 Max 固件无法通过 avrdude（通用烧录工具）或 Prusa Slicer 更新，多设备测试均失败，用户被迫使用“bug 原生固件”，设备沦为“摆设”（摘要 5）；Ender3 S1 Pro 非保机型升级固件需谨慎，内存卡需用 8G 正品，否则易变砖，新手无明确指引（摘要 1）。
- **显示与交互缺失**：K2 Plus LCD 屏幕不显示当前打印层数，仅能在创想云 App 查看，脱离手机无法掌握进度（摘要 3）。

## 三、共性衍生问题

- **新手友好度低**：软件无“新手引导”，固件升级/参数调整无清晰教程（如 Ender3 S1 Pro 升级需自行找官网视频），用户易因操作失误导致设备故障（摘要 1、4）；跨软件/跨机型适配差，用户切换设备时需重复调整参数，学习成本高（摘要 6）。
- **售后与优化滞后**：用户反馈的“打印撞模型”“固件响应延迟”等核心问题，长期无官方补丁，需依赖第三方社区解决方案（如 K2 Plus 的 CFS 热风干燥改造需用户自行刷固件）（摘要 3、5）。
